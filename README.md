# statistics

Flashcards for Mathematical Statistics — for spaced repetition review via [flashcards](https://github.com/thomasrribeiro/flashcards).

## Coverage

| File | Topic |
|------|-------|
| 01_descriptive | Measures of center/spread, quantiles, boxplots, skewness, kurtosis, correlation |
| 02_sampling_distributions | Statistic vs parameter, sample mean/variance distribution, $t$, $\chi^2$, $F$ |
| 03_point_estimation | Method of moments, maximum likelihood, score function, invariance |
| 04_estimator_properties | Bias, variance, MSE, consistency, efficiency, Cramér-Rao, sufficiency |
| 05_confidence_intervals | Pivots, CI for mean/variance/proportion, sample size, prediction intervals |
| 06_hypothesis_testing | Null/alternative, type I/II errors, power, $p$-values, Neyman-Pearson |
| 07_one_sample_tests | $z$-test, $t$-test, variance test, proportion test, exact binomial |
| 08_two_sample_tests | Two-sample $t$ (pooled/Welch), paired $t$, two-proportion, $F$ for variances |
| 09_chi_square_tests | Goodness-of-fit, independence, homogeneity, expected counts |
| 10_anova | One-way ANOVA, sums of squares, $F$ statistic, post-hoc, two-way intro |
| 11_simple_regression | OLS, assumptions, $R^2$, inference on slope, residual analysis |
| 12_multiple_regression | Multiple OLS, $F$-test, multicollinearity, adjusted $R^2$, model selection |
| 13_nonparametric_bayesian | Sign, Wilcoxon, Mann-Whitney, Kruskal-Wallis, bootstrap, Bayes intro |
